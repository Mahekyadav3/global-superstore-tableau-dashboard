Global Superstore Sales Insights Dashboard
An interactive business intelligence dashboard built with Tableau Public, analyzing global sales data from the Superstore dataset. This project showcases key sales metrics, regional performance, and product-level insights across multiple countries.

📌 Key Insights Covered

💰 Total Sales, Profit & Orders — High-level KPIs at a glance
🌏 Regional Sales Performance — Breakdown by region and country
📦 Category & Sub-Category Analysis — Which products drive the most revenue
📅 Sales Trends Over Time — Monthly and yearly patterns
🚚 Shipping Mode Analysis — Impact of shipping type on profit
